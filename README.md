# Real-Time Fraud Detection System

A synthetic behavioral transaction simulation and fraud detection system built using Python.

## Features

- Synthetic user generation
- Behavioral transaction simulation
- Fraud injection engine
- Multiple fraud patterns
- Transaction validation pipeline

## Fraud Types Simulated

- Velocity fraud
- Account draining
- Device takeover
- Impossible travel
- Card testing

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
